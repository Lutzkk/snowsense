# Package index

## All functions

- [`aggregate_snow()`](aggregate_snow.md) : Aggregates a binary snow
  mask to a fractional snow cover raster
- [`despeckle_snow()`](despeckle_snow.md) : Despeckles snow patches
  based on a binary snow mask (1 == snow, 0 == no snow)
- [`detect_snow()`](detect_snow.md) : Detects snow-covered areas from
  optical imagery using different spectral indices and brightness based
  approaches.
- [`example_s2_20m`](example_s2_20m.md) : Example Sentinel-2 RGB+SWIR
  raster (20 m)
- [`snow_cover_fraction()`](snow_cover_fraction.md) : Estimate Snow
  Cover Fraction (SCF) from NDSI
